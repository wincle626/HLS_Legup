Feedback:

If you have found a bug in GXemul, or if you have questions and you can't find the answers you seek in the documentation, then please send mail to one of the mailing lists:

  • gxemul-devel for (technical) discussions about the development of GXemul
  • gxemul-users for general user questions

or ask in the IRC channel #GXemul on FreeNode.
